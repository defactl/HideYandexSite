Вызов через консоль window.yandexFilter:  
// например, "yandexFilter.list()"  
toggleReveal() - переключатель глазика  
stats() - статистика  
list() - таблица в консоле индекс/домен  
export() - экспорт заблокированных сайтов формат .JSON  
import() - импорт заблокированных сайтов формат .JSON  
clear() - очисть весь список заблокированных сайтов  
remove() - удалить отдельных сайт из заблокированных сайтов  
add() - добавить сайт в заблокированные сайты
